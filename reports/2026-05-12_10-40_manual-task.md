2. Open the Electron window on Windows.
3. Go to Settings and confirm there are no Whisper binary/model path fields.
4. Enter a fake API key, a valid HTTPS base URL, a correction model, a summary model, and a save folder.
5. Save settings and confirm the success state appears.
6. Open the Rubai setup doc and confirm the converted model expectations match the local folder.
7. Go to Book, import a `.txt` file, then search for a known term and confirm snippets appear.
8. Go to Dashboard, start capture, and confirm later chunks stay `pending` while only one chunk is `transcribing`.
9. Confirm failed chunks show a chunk-specific error and can be retried.
