## Manual UI Check

1. Run `npm run dev`.
2. Open the Electron window.
3. Go to Settings and configure the required OpenAI API key, Whisper binary path, and Whisper model path fields.
4. Go to Dashboard.
5. Select a desktop source.
6. Confirm the Dashboard shows a transcript workspace with two editable panes: Raw transcript and Corrected transcript.
7. Type manual text into the Raw transcript pane.
8. Type different manual text into the Corrected transcript pane.
9. Switch to another tab and back to Dashboard.
10. Confirm both manual edits are still visible.
11. Start recording.
12. Wait until at least one chunk finishes transcribing locally.
13. Confirm the finished chunk text appends to the Raw transcript pane without replacing existing manual edits.
14. Confirm the Corrected transcript pane remains editable and is not changed by local chunk transcription.
15. Stop recording.
