## Implementation Summary

- Added a main-process Whisper config validation helper that checks binary and model paths before transcription or recording starts.
- Exposed `validateWhisperConfig` through IPC and the preload bridge.
- Added a Settings action to validate the configured Whisper binary and model paths without starting a recording.
- Added recorder preflight validation so invalid Whisper config fails before `MediaRecorder` capture starts.
- Normalized pasted Windows paths wrapped in quotes and continued passing Whisper CLI args as an argument array so paths with spaces are handled safely.
- Improved Whisper failure messages to prefer captured `stderr` and include recent process output in user-facing errors.

## Manual UI Check

1. Run `npm run dev`.
2. Open the Electron window.
3. Go to Settings.
4. Enter a fake OpenAI API key so required field validation is not blocking save.
5. Enter a Whisper binary path with spaces, such as a path under `C:\Program Files\...`, and a valid local model path.
6. Click `Validate Whisper config`.
7. Confirm Settings shows `Whisper binary and model paths are valid.`
8. Wrap the same valid binary or model path in double quotes and click `Validate Whisper config` again.
9. Confirm validation still succeeds.
10. Change the Whisper binary path to a missing file and click `Validate Whisper config`.
11. Confirm the error names `Whisper binary path` and shows the missing path.
12. Change the Whisper model path to a missing file and click `Validate Whisper config`.
13. Confirm the error names `Whisper model path` and shows the missing path.
14. Go to Dashboard.
15. Select a desktop source.
16. Keep an invalid Whisper binary or model path configured.
17. Click Start.
18. Confirm recording does not start and the recorder shows a useful error before capture begins.
19. Restore valid Whisper paths.
20. Start recording and confirm chunks can proceed to local transcription.
21. Trigger a Whisper failure, for example with an incompatible model file, and confirm the displayed error includes useful `stderr` text when Whisper emits it.
