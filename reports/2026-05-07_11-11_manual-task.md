## Manual UI Check

1. Run `npm run dev`.
2. Open the Electron window.
3. Go to Settings.
4. Configure a local Whisper binary path and local Whisper model path.
5. Go to Dashboard.
6. Select a desktop source that has audio.
7. Click Start.
8. Confirm the active chunk appears with status `recording`.
9. Wait for the chunk interval to finish.
10. Confirm the completed chunk moves to `transcribing`.
11. Confirm a successful chunk moves to `done`.
12. Confirm the raw transcript editor receives the chunk transcript.
13. Type a manual edit into the raw transcript editor.
14. Record another chunk and confirm the new transcript is appended without deleting the manual edit.
15. Temporarily set an invalid Whisper model path and record a chunk.
16. Confirm the chunk moves to `failed`.
17. Restore the correct Whisper model path if needed.
18. Click Retry on the failed chunk.
19. Confirm retry changes the chunk back to `transcribing`, then `done` if Whisper succeeds.
20. Confirm no raw audio is sent to OpenAI and no OpenAI audio API call exists in the flow.
